✅変数名とオブジェクトのキーの扱い方の違い

オブジェクトのキーは参照される
変数は変数に代入された値のみが参照され、変数名は参照されない

例えば

export async function　関数（）｛
処理省略
const result = {
  a:1
  b:yamada
  c:サッカー
}
return{
  data:result
}
}


返される値は
data:{
  a:1
  b:yamada
  c:サッカー
}
となる
data:{
  result{
  a:1
  b:yamada
  c:サッカー
  }
}
とならない。

関数に出力先でも
const {data} = 関数()
data:{
  a:1
  b:yamada
  c:サッカー
}
となる
オブジェクトのキー名dataは残るが、変数名resultは残らない。
変数は中身のみが参照される
