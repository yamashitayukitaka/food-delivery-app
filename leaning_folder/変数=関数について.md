✅const 変数　= 関数
この場合通常関数で取得した値を変数にいれる
1.まず右辺の関数実行
2.右辺で取得できた値を変数に格納

例1．
const response = await fetch(url)
1.右辺のfetch実行
2.右辺のfetchで得られた値を変数に格納

例2．
◎<?php $title = get_the_title?> 
のように取得用の関数を実行して得られた値を複数個所で表示するために 上記のような式は使う

✖ <?php $title = the_title?> のように値を返すのではなく出力するための関数は変数には入れられない
変数＝関数の場合 関数の返り値を格納するためのであるから、出力用の関数であり値を取得する為の関数ではない
<?the_title;?>は変数　= 関数のかたちでは使えない
