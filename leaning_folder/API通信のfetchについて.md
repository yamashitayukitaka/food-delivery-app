💡fetch() の役割は 2つ
const response = await fetch(url)

① リクエストの送信（送る側）
fetch() は、HTTPメソッド（GET, POST, PUT, DELETEなど）に従って
指定したURLに リクエスト を送ります。
例：POST ならデータ送信、GET ならデータ取得など。


② レスポンスの受信（受け取る側）
fetch() は、サーバー（APIなど）からの レスポンス を受け取り、
それを表す Responseオブジェクト を返します。
そのため、受け取った内容を使うにはさらに
const data = await response.json();
のようにして中身を取り出します。


🔁 POST と GET の違い
メソッド	主な用途	サーバーからデータが返るか
GET	データ取得	通常、レスポンスとしてJSONやHTMLなどのデータを返す
POST	データ送信	通常は「成功したかどうか」などの結果だけ返す（データは返さないことが多い）


🧩ただし、例外的に：
Google Places API のようなものでは、
POSTで検索条件（例：位置情報やキーワード）を送ると、
同時に結果データ（場所一覧）を返す 仕様になっています。
これは「POSTしてデータも返す」という特殊なパターンです。



✅const 変数　= 関数
この場合通常関数で取得した値を変数にいれる
1.まず右辺の関数実行
2.右辺で取得できた値を変数に格納

例1．
const response = await fetch(url)

2.右辺のfetchで得られた値を変数に格納
